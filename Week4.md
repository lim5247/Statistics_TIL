# 1️⃣ 개념 정리 
## 01. 양적 변수 사이의 관계를 밝히다

```
✅ 학습 목표 :
* 상관과 회귀에 대해 이해한다.
```
**산점도**<br>
두 변수를 x, y축에 할당하여 각각의 데이터를 점으로 표현한 그래프  

**상관** 2개의 확률변수 사이의 관계성, 인과관계는 알 수 없음
**회귀**  
- 함수를 통해 변수 사이의 관계를 공식화
- x: 독립변수, 설명변수
- y: 종속변수, 반응변수

## 02. 상관관계

```
✅ 학습 목표 :
* 피어슨 상관계수의 개념과 특징에 대해 설명할 수 있다.
* 비모수 상관계수에 대해 설명할 수 있다.
* 상관계수의 유의성 검정과 p값·표본크기의 영향을 이해한다.
```

**상관관계 분석 하기 전**  
1. x, y축의 변수는 개별 변수여야 함
2. 정규성이 성립되는지 확인해야 함

**피어슨 상관계수 r** -> 정규성 성립했을 때  
- 상관계수의 대표격
- 양적 데이터의 선형관계
- -1 =< r =< 1
- 분자는 공분산, 분모는 각 변수의 표준편차 곱
- 따져야 할 것들
  - 부호: 양의상관관계, 음의상관관계
  - 절댓값: 1에 가까울 수록 직선에 가까워짐
    - 0.7 < |r| =< 1: 강한 상관
    - 0.4 < |r| =< 0.7: 중간 정도 상관
    - 0.2 < |r| =< 4: 약한 상관
    - 0.0 < |r| =< 2: 거의 상관없음
- 선형관계를 이룸 -> 다른 분포 못 반영
- 기울기의 크기는 관계가 없음
- 산점도를 미리 그려 분포를 확인해야함
<img width="590" height="201" alt="image" src="https://github.com/user-attachments/assets/29aaa132-3209-4629-bb4f-d81fc4ba30c6" />

**비모수 상관계수** -> 정규성 성립하지 않았을 때  
**[스피어만 순위상관계수 ρ]**  
- 순위로 변환한 값의 선형관계
- -1 =< ρ =< 1
- 이상값이 있을 때도 사용 가능  
**[켄달 순위상관계수 τ]**
- 표본크기 n이 매우 작을 때 유의성검정에서 더 좋음  

**상관계수와 가설**  
모집단 상관계수 rρ <- 표본 상관계수 r
- 귀무가설: rρ = 0
- 대립가설: rρ ≠ 0  
* p < 0.05 이면 대립가설 채택
* 표본크기가 크면 귀무가설에 어긋나더라도 통계적으로 유의미하다고 판단될 수 있다

**비선형 상관**  
상관이라는 것은 x, y가 서로에 관해 어느 정도의 정보를 포함한다는 것  
이런 의미에서 비선형 상관관계도 포함될 수 있음

## 문제 1.

🧚 **Q. 다음은 다중회귀분석 결과이다.**

한 연구자가 학생들의 공부시간(X₁), 수면시간(X₂), 학원 수강 여부(0=미수강, 1=수강)를 설명변수로 하여 시험 점수(Y)를 예측하는 다중회귀모형을 분석하였다.

분석 결과는 다음과 같다.

- 공부시간의 회귀계수: **2.5** (p < 0.01)  
- 수면시간의 회귀계수: **-0.8** (p = 0.20)  
- 학원 수강 여부의 회귀계수: **5.0** (p < 0.05)
