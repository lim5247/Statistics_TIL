# 1️⃣ 개념 정리 
## 01. 양적 변수 사이의 관계를 밝히다

```
✅ 학습 목표 :
* 상관과 회귀에 대해 이해한다.
```
**산점도**<br>
두 변수를 x, y축에 할당하여 각각의 데이터를 점으로 표현한 그래프  

**상관** 2개의 확률변수 사이의 관계성, 인과관계는 알 수 없음
**회귀**  
- 함수를 통해 변수 사이의 관계를 공식화
- x: 독립변수, 설명변수
- y: 종속변수, 반응변수

## 02. 상관관계

```
✅ 학습 목표 :
* 피어슨 상관계수의 개념과 특징에 대해 설명할 수 있다.
* 비모수 상관계수에 대해 설명할 수 있다.
* 상관계수의 유의성 검정과 p값·표본크기의 영향을 이해한다.
```

**상관관계 분석 하기 전**  
1. x, y축의 변수는 개별 변수여야 함
2. 정규성이 성립되는지 확인해야 함

**피어슨 상관계수 r** -> 정규성 성립했을 때  
- 상관계수의 대표격
- 양적 데이터의 선형관계
- -1 =< r =< 1
- 분자는 공분산, 분모는 각 변수의 표준편차 곱
- 따져야 할 것들
  - 부호: 양의상관관계, 음의상관관계
  - 절댓값: 1에 가까울 수록 직선에 가까워짐
    - 0.7 < |r| =< 1: 강한 상관
    - 0.4 < |r| =< 0.7: 중간 정도 상관
    - 0.2 < |r| =< 4: 약한 상관
    - 0.0 < |r| =< 2: 거의 상관없음
- 선형관계를 이룸 -> 다른 분포 못 반영
- 기울기의 크기는 관계가 없음
- 산점도를 미리 그려 분포를 확인해야함

**비모수 상관계수** -> 정규성 성립하지 않았을 때  
<br>
**[스피어만 순위상관계수 ρ]**  
- 순위로 변환한 값의 선형관계
- -1 =< ρ =< 1
- 이상값이 있을 때도 사용 가능

**[켄달 순위상관계수 τ]**
- 표본크기 n이 매우 작을 때 유의성검정에서 더 좋음  

**상관계수와 가설**  
모집단 상관계수 rρ <- 표본 상관계수 r
- 귀무가설: rρ = 0
- 대립가설: rρ ≠ 0  
* p < 0.05 이면 대립가설 채택
* 표본크기가 크면 귀무가설에 어긋나더라도 통계적으로 유의미하다고 판단될 수 있다

**비선형 상관**  
상관이라는 것은 x, y가 서로에 관해 어느 정도의 정보를 포함한다는 것  
이런 의미에서 비선형 상관관계도 포함될 수 있음

## 03. 선형회귀
```
✅ 학습 목표 :
* 최소제곱법의 원리를 이해한다. 
* 회귀계수의 가설검정에 대해 설명할 수 있다.
* 결정계수에 대해 설명할 수 있다.
* 오차의 등분산성과 정규성에 대해 설명할 수 있다.
* 설명변수와 반응변수에 대해 설명할 수 있다.
```
**회귀식** 설명변수와 반응변수 사이의 관계성을 알 수 있는 함수
- 설명변수가 1개면 단순회귀, 여러 개면 다중회귀
- 회귀계수: 회귀식의 형태를 결정하는 파라미터

* 회귀분석의 흐름
1. 회귀의 적함도 평가
2. 회귀계수의 값을 구체적으로 구하기

**최소제곱법**  
- 회귀계수를 결정하는 방법
- 잔차 제곱의 합이 최소가 되도록 하는 회귀 계수를 찾아가는 방법
  - 잔차: 실제 데이터값과 회귀식으로 예측된 값의 차이

**회귀계수의 가설검정**  
- 회귀분석을 통해 얻은 기울기가 통계적으로 의미가 있는지 확인하는 과정
- 귀무가설: 기울기 b = 0, 설명변수 사이에는 아무런 선형관계가 없다
- p < 0.05이면 귀무가설 기각 -> 유의미한 선형관계가 있다고 판단
- 그러나 상관관계일뿐 인과관계를 의미하지는 않음

**결정계수 R²**  
- 회귀식의 적합도 판단, 전체 분산 중 회귀모형에 의해 설명되는 비율을 나타냄(0~1)
- 1에 가까울 수록 데이터를 완벽하게 설명
- 설명변수의 개수가 많으면 실제보다 설명력이 향상된 것처럼 보임 -> 조정 설명계수 사용

**확률오차의 등분산성과 정규성**  
- 회귀계수의 가설검정이나 신뢰구간을 구하기 위해 필요한 기본 가정
* 정규성: 오차항의 분포가 평균 0인 정규분포를 따라야 함, 잔차의 샤피로-윌크 검정
* 등분산성: 설명변수값과 상관없이 오차의 분산이 일정해야 함, 브루쉬-페이건 검정

**설명변수와 반응변수**  
상관분석과 달리 인과관계가 존재
- 설명하는 쪽을 설명변수로, 설명할 대상을 반응변수로


## 04. 선형회귀 원리의 확장

```
✅ 학습 목표 :
* 다중회귀를 이해하고 그 결과를 읽을 수 있다.
* 편회귀계수의 의미를 이해하고 해석할 수 있다.
* 범주형 변수를 더미변수로 변환하는 방법과 회귀계수의 해석을 이해한다.
* 공분산분석에 대해 설명할 수 있다.
* 다중공선성에 대해 이해한다.
```
**다중회귀** 여러 개의 설명변수로 반응변수를 설명하는 모델  
- 회귀평면: 설명변수가 2개 일떄 회귀모형은 평면으로 나타남

**편회귀계수**  
- 다중회귀 모델 함수에서 기울기를 의미함
- 다른 변수를 고정한 채 해당 변수가 1단위가 증가할 때 y가 변화하는 양

**범주형 변수를 더미변수로 변환하기**  
더미변수: 범주형 변수를 회귀분석에 넣기 위해 0 또는 1의 값을 가진 가변수를 만들 수 있음
- 설명변수를 (범주개수 - 1)개 만큼 설정
  - ex) A, B, O, AB -> {0, 0, 0}, {1, 0, 0}, {0, 1, 0}, {0, 0, 1} ({} 안에 변수x 3개)

**공분산 분석(ANCOVA)**  
- 분산분석(집단 간 평균 비교)에 양적 변수(공변량) 추가하여 분석 정밀도를 높이는 방법
- 목적: 반응변수에 영향을 주는 제 3의 요인(공변량)의 효과를 배제하고 순수하게 집단 간의 차이만을 보기 위함
* 집단 간 회귀직선의 기울기가 서로 비슷해야 함
* 공변량의 회귀계수가 0이 아니어야 함

**다중공선성**  
- 설명변수 사이의 강한 상관관계가 있어 회귀계수 추정에 신뢰성이 떨어지는 문제
- VIF(분산 팽창 인수)를 계산 하여 확인, VIF > 10이면 심각한 다중공선성 상태
- 상관이 너무 높은 변수 하나 삭제 or 주성분분석 등의 차원 축소 방법 사용으로 해결 가능
