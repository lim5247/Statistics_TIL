# 1️⃣ 개념 정리 
## 01. 회귀모형의 형태 바꾸기

```
✅ 학습 목표 :
* 이원배치 분산분석에 대해 설명할 수 있다.
* 비선형회귀에 대해 설명할 수 있다.
```
**상호작용**  
- 상호작용: 설명변수 간 상승효과, 선형회귀모형에서 곱셈으로 표현
- 상호작용항 적용하기 좋은 경우  
  - 상호작용이 있다는 것이 선행연구에서 밝혀지거나 기대되는 때  
  - 데이터에 분명한 상호작용이 있을 때  
  - 상호작용 유무에 관심이 있을 때
 
<img width="749" height="76" alt="image" src="https://github.com/user-attachments/assets/15469b40-4afb-440f-a367-b6ee315355ae" />


**이원배치 분산 분석** -> 범주형 변수 대상  
- 일원배치 분산 분석 vs 다원배치 분산 분석
- 설명변수 간 상호작용 문제
  - 상호작용항 c가 유의미하지 않을 때  
    -> 상호작용 없다고 보고 주효과를 그대로 평가, 한 설명변수의 각 범주 당 기울기 평행
  - 상호작용항 c가 유의미할 때  
    -> 각각의 요인을 나누어 주효과를 하위 검정으로 평가, 본페로니 교정 등의 다중비교 이용 -> 유의수준 조정, 한 설명변수의 각 범주 당 기울기 평행 X

**비선형 회귀**  
- x에 관한 비선형 모델 (이 경우에도 파라미터에 관해서 1차식이라면 선형모델이라고 함)
  - 해석이 목적이라면 보통 1차함수의 회귀모형을 이용함
  - 예측이 목적이면 모형의 복잡도를 올려도 괜찮음  

- 파라미터에 관한 비선형회귀모형
  - 왜 그 모형을 사용하는지 합리적인 이유가 있어야 함  
    ex) 미하엘리스-멘텐식 ... 분모에 파라미터가 있어 비선형모형 but 효소반응에 잘 맞음  

- 비선형 회귀 주의할 점
  - 비선형 회귀에서 최소제곱법으로 파라미터를 구할 수 없음, 컴퓨터 필요
  - 국소 최적해 주의


## 02. 일반화선형모형의 개념

```
✅ 학습 목표 :
* 선형회귀가 적절하지 않은 상황을 판단할 수 있다.
* 가능도와 최대가능도 방법을 이해한다.
* 로지스틱 회귀와 오즈비에 대해 이해한다.
* 푸아송 분포와 푸아송 회귀에 대해 이해한다.
* 다양한 일반화선형모형에 대해 설명할 수 있다.
```

**선형회귀 원리 확장하기**  
- 일반선형모형: 최소제곱법을 이용하여 파라미터 추정을 함
  - 반응변수가 양적변수
  - 오차분포가 정규분포
  - 설명변수에 따라 오차분산이 변하지 않음  
  -> 그렇지 않으면 직선모델이 부적절, 해석 어려움

- 일반화선형모델: 최대가능도방법으로 회귀모형 추정, 데이터 성질에 맞는 확률모형(이항분포, 푸아송 분포, 베이즈 추정 등)을 가정하여 유연한 모형화가 가능

**가능도와 최대가능도 방법**  
- 가능도(우도): 파라미터 $\theta$가 주어졌을 때 관찰된 데이터 x가 나타날 확률
- 최대 가능도 방법: 가능도를 최대화하는 $\theta$를 찾는 것
  - 계산의 편의를 위해 로그를 취할 수 있음
<img width="798" height="84" alt="image" src="https://github.com/user-attachments/assets/2ce8df70-1f36-4532-acaf-183b273c886f" />

**로지스틱 회귀**  
<img width="740" height="504" alt="image" src="https://github.com/user-attachments/assets/d8ef0663-739f-4494-a83e-f2a61ad33d77" />
<br>
- 반응변수가 두 가지 값만 가지는 변수형일 떄
- 이항분포를 가정하여 모델링
- 로지스틱 함수: 그림의 좌변(선형예측변수)을 p로 변환시키는 함수
  - a는 평행이동, b는 변화정도
- 로짓함수: 그림의 우변, 연결함수
- 오즈: 어떤사건이 일어날 확률p와 일어나지 않을 확률 1-p의 비율
- 오즈비(OR): 두 집단의 오즈를 비교한 비율, 1보다 크면 분자 집단의 사건이 더 잘 일어난다는 뜻
  - 로지스틱 회귀의 결과값인 회귀계수(b)를 오즈비로 전환 가능, 수학적 취급이 용이
<img width="692" height="252" alt="image" src="https://github.com/user-attachments/assets/449a7782-d838-4d11-b298-770d6df7a090" />

**푸아송 회귀**
- 반응변수가 0이상의 정수일 때, 특히 반응변수가 개수 일 때
- 파라미터가 평균($\lambda$) 하나뿐
- 평균 = 분산: 서로 비독립적
- 확률모델(오차구조): 평균이 $\lambda_i$인 푸아송 분포
- 회귀식: $\lambda_i = exp(a + bx_i)$
- 연결함수: $\log(\lambda_i) = a + bx_i$

**다양한 일반화 선형 모형**
| 확률분포       | 반응변수 (Y) 성질      | 연결함수           | 주요 특징 및 용도 |
|----------------|------------------------|--------------------|------------------|
| 이항분포       | 0 이상 정수 (상한 있음) | logit             | 0또는 상한이 있는 개수 데이터 |
| 푸아송 분포    | 0 이상의 정수          | log                | 상한이 없는 개수 데이터, 평균=데이터 |
| 음이항 분포    | 0 이상의 정수          | log / inverse      | 상한이 없는 개수 데이터, 평균<분산 |
| 정규분포       | 실수       | (미사용)  | 범위에 제한이 없음, 분산은 일정 |
| 감마분포       | 0 이상의 정수          | log / inverse      | 분산이 변함 |
