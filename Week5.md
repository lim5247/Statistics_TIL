## 02. 일반화선형모형의 개념

```
✅ 학습 목표 :
* 선형회귀가 적절하지 않은 상황을 판단할 수 있다.
* 가능도와 최대가능도 방법을 이해한다.
* 로지스틱 회귀와 오즈비에 대해 이해한다.
* 푸아송 분포와 푸아송 회귀에 대해 이해한다.
* 다양한 일반화선형모형에 대해 설명할 수 있다.
```

**선형회귀 원리 확장하기**  
- 일반선형모형: 최소제곱법을 이용하여 파라미터 추정을 함
  - 반응변수가 양적변수
  - 오차분포가 정규분포
  - 설명변수에 따라 오차분산이 변하지 않음  
  -> 그렇지 않으면 직선모델이 부적절, 해석 어려움

- 일반화선형모델: 최대가능도방법으로 회귀모형 추정, 데이터 성질에 맞는 확률모형(이항분포, 푸아송 분포, 베이즈 추정 등)을 가정하여 유연한 모형화가 가능

**가능도와 최대가능도 방법**  
- 가능도(우도): 파라미터 $\theta$가 주어졌을 때 관찰된 데이터 x가 나타날 확률
- 최대 가능도 방법: 가능도를 최대화하는 $\theta$를 찾는 것
  - 계산의 편의를 위해 로그를 취할 수 있음

**로지스틱 회귀**

- 반응변수가 두 가지 값만 가지는 변수형일 떄
- 이항분포를 가정하여 모델링
- 로지스틱 함수: 그림의 좌변(선형예측변수)을 p로 변환시키는 함수
  - a는 평행이동, b는 변화정도
- 로짓함수: 그림의 우변, 연결함수
- 오즈: 어떤사건이 일어날 확률p와 일어나지 않을 확률 1-p의 비율

  - 로지스틱 회귀의 결과값인 회귀계수(b)를 오즈비로 전환 가능, 수학적 취급이 용이

**푸아송 회귀**
- 반응변수가 0이상의 정수일 때, 특히 반응변수가 개수 일 때
- 파라미터가 평균($\lambda$) 하나뿐
- 평균 = 분산: 서로 비독립적
- 확률모델(오차구조): 평균이 $\lambda_i$인 푸아송 분포
- 회귀식: $\lambda_i = exp(a + bx_i)$
- 연결함수: $\log(\lambda_i) = a + bx_i$

**다양한 일반화 선형 모형**
| 확률분포       | 반응변수 (Y) 성질      | 연결함수           | 주요 특징 및 용도 |
|----------------|------------------------|--------------------|------------------|
| 이항분포       | 0 이상 정수 (상한 있음) | logit             | 0또는 상한이 있는 개수 데이터 |
| 푸아송 분포    | 0 이상의 정수          | log                | 상한이 없는 개수 데이터, 평균=데이터 |
| 음이항 분포    | 0 이상의 정수          | log / inverse      | 상한이 없는 개수 데이터, 평균<분산 |
| 정규분포       | 실수       | (미사용)  | 범위에 제한이 없음, 분산은 일정 |
| 감마분포       | 0 이상의 정수          | log / inverse      | 분산이 변함 |

## 04. 통계 모형의 평가와 비교

```
✅ 학습 목표 :
* 왈드 검정과 가능도비 검정에 대해 설명할 수 있다.
* AIC와 BIC에 대해 설명할 수 있다.
```
**왈드 검정** 최대 가능도 추정값을 표준오차로 나눈 왈드통계량을 이용
- 추정량이 정규분포를 따른 다고 가정 -> 계산 간편
- 표본 크기가 작을 때는 정규분포 가정이 불확실해 신뢰도가 낮아짐

**가능도비 검정** 두 모형 간의 최대가능도 비율을 비교하여 개선 정도를 확인
- 비교할 두 모형이 내포 관계여야함
- 왈드 검정보다 신뢰도가 높은 편임

**AIC** 새로운 데이터를 얼마나 잘 예측하는지 평가
- $AIC = -2 \log L^* + 2k$ (k: 파라미터 개수)
- 파라미터가 늘어날수록 패널티를 주어 과대적합을 방지
- AIC를 최소화하는 모형을 선택
- 과대적합: 실제 데이터에 무리하게 맞추다가 새롭게 얻은 데이터는 제대로 나타내지 못하는 상태

**BIC** 실제 모형 탐색이 목적, 후보 중 진짜 모형일 확률이 높은 것을 선택
- $BIC = -2 \log L^* + k \log(n)$ (n: 표본 크기)
- 표본 크기가 클수록 파라미터 증가에 대해 AIC보다 엄격한 패널티를 부여
- BIC를 최소화하는 모형을 선택

**AIC VS BIC**
* AIC: 복잡한 현상에 대해 여러 개의 후보 중 실제 모형이 있다고 상정할 수 없는 경우
* BIC: 비교적 간단한 현상에 대한 후보중 올바른 것으로 보이는 하나를 선택하는 경우

  - p-해킹: 연구자가 의도적 혹은 무의식적으로 유의하도록 실험 조작

## 06. 가설검정의 문제점

```
✅ 학습 목표 :
* 왜 유의수준 α = 0.05를 사용하는지 설명할 수 있다.
* 피셔류 검정과 네이만-피어슨류 검정 체계의 이론적 차이와 적용 목적을 설명할 수 있다.
* 효과크기를 이해한다.
* 베이즈 인수를 이해한다.
```

**유의수준 α = 0.05의 유래**  
- 사용근거: 관례 -> 귀무가설을 기각했으나 실제로는 귀무가설이 옳았던 경우가 많음

**피셔류 검정과 네이만-피어슨류 검정**  
- 피셔류 검정: 귀무가설이 옳을 때 관찰된 데이터가 나타날 확률인 p값 계산 -> 귀무가설과 관찰값의 괴리 파악
- 네이만-피어슨 검정: p값이 유의수준 a 미만인지 파악하여 가설 채택 여부 결정

**효과 크기**  n의 크기와 관련없이 실제 차이가 얼마나 큰지 확인해야 함
- Cohen's d: 두 집단의 평균값 차이를 모집단의 표준편차로 표준화한 지표
- 피어슨 계수: 상관관계, 선형적 관계의 강도
- 분산분석: 전체 변동 중 특정 요인이 설명하는 비율
- 결정계수 R^2: 회귀분석에서 설명력을 드러냄

**베이즈 인수** p값 대신 사용, 주변 가능도의 비
$$p(x|M) = \int p(x|\theta, M) p(\theta|M) d\theta$$
- 2가지 가설을 대등하게 비교, 귀무가설 적극 지지 가능
- 한쪽 가설을 강하게 지지할 때까지 표본을 계속 늘려가는 순차적 갱신 가능
- ㅎ한쪽 가설이 나쁜 것만으로 베이즈 인수가 큰값이 될 수도 있음
- 베이즈 인수는 세타의 사전 분포에 영향을 크게 받음
- p값에 비해 시간이 많이 걸림

~~~

4️⃣ p값이 0.05보다 작으므로 과학적으로 효과가 증명되었다.
~~~
