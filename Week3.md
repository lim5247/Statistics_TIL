# 1️⃣ 개념 정리 
## 01. 가설검정의 원리

```
✅ 학습 목표 :
* 가설검정의 원리에 대해 설명할 수 있다.
* 통계적에서의 가설을 이해한다.
```
**가설 검정의 정의**  
- 정의: 가설이 타당한지 P-value를 이용하여 판단하는 추론통계 방식

**자료 분석의 두 가지 접근법**  
- 확증적 자료분석: 미리 가설을 세우고 검증하기 위해 데이터 수집 분석  
- 탐색적 자료분석: 미리 가설을 세우지 않고 데이터의 특징 파악 및 가설 후보 선정을 위한 데이터 해석 방식  

**통계학적 가설의 종류**  
- 귀무가설(H0, Null Hypothesis): 밝히고 싶은 가설의 부정명제  
- 대립가설(H1, Alternative Hypothesis): 밝히고 싶은 가설  

**가설검증의 논리 구조(귀류법)**  
-> 밝히고자하는 명제가 잘못되었다는 것을(귀무가설이 맞다고) 가정하고 모순을 발견하는 방식  
* 표본 상의 차이가 포본오차로 인해 발생한 것은 아닌지 확인해야함  

**P값과 유의수준(a)**
- P값: 귀무가설이 옳다는 가정 하에 관찰된 값 이상으로 극단적인 값이 나올 확률, P값이 일정값 이하로 나온다면 그 차이는 귀무가설을 기각하고 대립가설을 채택할 만큼 '통계적으로 유의미함'
- 유의수준(a): 가설 판정의 경계가 되는 값, 일반적으로 0.05를 기준으로 삼음

## 02. 가설검정 시행 

```
✅ 학습 목표 :
* p값과 기각역에 대해 설명할 수 있다.
* 가설검정의 구체적인 예를 이해한다.
```
1. 검정 통계량 계산
2. P값 또는 기각역으로 귀무가설의 기각과 채택을 결정
- 양측검정: 양수 음수 모두 고려, 일반적
- 단측검정: 한쪽만 고려하여 계산
* p값이 작으면 귀무가설 기각
* 기각역에 검정통계량이 해당되면 귀무가설 기각  

참고) 이표본 t검정: 2개 집단 간의 평균을 비교하는 검정 방식  

## 03. 가설검정 관련 그래프
```
✅ 학습 목표 :
* 오차막대에 대해 설명할 수 있다.
* '통계적으로 유의미'의 의미를 이해한다.
```
**오차막대**  
- 평균값의 확률 표현: [평균값 ± 표준오파(SEM)] 사용, 이때 표준오차는 s/루트n
- 신뢰구간 표현:  평균값을 중심으로 95% 신뢰구간 그림
- 데이터 퍼짐 표현: [평균값 ± 표준편차] 사용, 평균값의 확률 아님!!
* 오차막대가 무엇을 나타내는지는 따로 기재해야함

**통계적으로 유의미**  
- 통계적으로 유의미하다: 우연이라고 보기 어려움, *을 이용하여 표현
- 통계적으로 유의미하지 않다: N.S.

## 04. 제1종 오류와 제2종 오류

```
✅ 학습 목표 :
* 제1종 오류와 제2종 오류의 개념을 구분하고 설명할 수 있다.
* 유의수준 α와 제1종 오류의 관계를 이해한다.
* 제2종 오류 β와 검정력(1-β)의 의미를 이해한다.
* α와 β의 상충 관계를 설명할 수 있다.
* 표본크기와 효과크기가 검정력에 미치는 영향을 이해한다.
```

**제 1종 오류와 제 2종 오류**
* 제 1종 오류: 귀무가설이 옳은데 귀무가설 기각, 유의수준 a
* 제 2종 오류: 귀무가설이 틀린데 귀무가설 채택, 대립가설 기각  
- 베타: 제 2종 오류가 일어나는 확률
- 검정력: 1 - 베타, 일반적으로는 80%로 설정, 통제는 불가  

**알파와 베타의 관계**  
알파와 베타는 서로 상충관계: 하나가 커지면 다른게 작아짐  

**표본크기와 효과크기가 검정력에 미치는 영향**  
- 효과크기(d): 일반적으로 얼마나 큰 효과가 있는지, 표준편차가 클수록 효과크기는 작아짐
- 표본크기(n): 표본크가 과도하게 크면 실제로 미미한 차이이더라도 통계적으로 유의미하다고 나올 수도 있음

## 06. 대푯값 비교

```
✅ 학습 목표 :
* 모수 검정의 평균값 비교에 대해 설명할 수 있다.
* 적절하지 않은 검정을 사용하면 어떻게 되는지 설명할 수 있다.
* 비모수 검정의 대푯값 비교에 대해 설명할 수 있다.
* 자유도와 분산분에 대해 설명할 수 있다.
* 다중비교 검정에 대해 설명할 수 있다.
```

**모수검정의 평균값 비교**  
* 대응 관계가 없는 검정: 두 집단의 피험자가 서로 다를 때, 독립
  * 일표본 t검정: 단일 모집단 평균이 특정 기준값과 같은지 판단, 등분산성 가정
  * 이표본 t검정: 서로 다른 두 집단의 평균 차이를 비교, 등분산성 가정
  * 웰치의 t검정: 분산이 다를 때, 정규성은 필요
* 대응 관계가 있는 검정: 선후 관계처럼 피험자가 같거나 관련이 있을 때
    * 일표본 t검정: '차이 평균값 = 0' 이 귀무가설
 
**정규성 조사**  
* q -q 플롯: 시각적 판단
* 샤피로 윌크검정: 가설검정을 이용한 조사
* 콜모고로프-스미르노프 검정(K-S): 이론적 분포와 비교

**등분산성 조사**  
바틀렛 검정, 레빈검정

**비모수 검정의 대표값 비교**  
- 정규성이 없는 경우 -> 비모수 검정
- 평균값X, 분포의 위치를 나타내는 대표값 중요
- 윌 콕슨 순위합 검정, 맨-휘트니 U검정
- 정규분포가 아니더라도 분포 모양이 같아야함, 등분산
- if) 세 집단 이상 -> 크러스컬-윌리스 검정, 스틸-드와스 검정, 스틸 검정

**분산 분석**  
* ANOVA 분석: 3 개집단 이상의 평균값 비교
  * 귀무가설: 모든 집단의 평균이 같다
  * 대립가설: 적어도 한 쌍에는 차이가 있다
 
**다중비교검정**  
* 다중성 문제: 집단이 여러 개인 경우 집단이 여러 개일 수록 제 1종 오류가 발생할 가능성 높음
  -> 다중 비교 검정 필요, 유의 수준을 엄격하게!

  * 본페로니교정: 검정 반복 횟수 k로 알파를 나눈 수를 기준으로
  * 튜키 검정: 본페르니 교정 보완, 모든 쌍 비교
  * 던넷 검정: 대조군과의 비교에만 관심이 있는 경우
  * 윌리엄스 검정: 집단 간의 순위를 매길 수 있는 경우
