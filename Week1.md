# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```

**데이터와 통계학**  
데이터: 수치의 모음  
통계학: 데이터를 해석하는 것

**데이터 분석의 목적**
1. 데이터 요약 ex) 평균값계산
2. 대상 설명(성질, 관계성)
  관계성의 종류  
  인과관계: 원인과 결과, 원인을 바꾸어(개입하여) 원하는 결과로  
  상관관계: 한쪽이 크면 다른쪽도 큼, 변화X
   *실험군, 대조군(통제군) 설정의 중요성  
3. 미지의 데이터 예측 ex) 선형관계, 기계학습
   이미 얻은 데이터를 바탕으로 새로운 데이터 예측
   최대한 오차가 작게

## 02. 통계학의 역할

```
✅ 학습 목표 :
* 산포에 대해 설명할 수 있다.
```

**통계학은 데이터 퍼짐 정도가 클수록 힘을 발휘한다**  
퍼짐(산포, dispersion) -> 대상의 성질이나 관계성을 감춤  
=> 확률(론)을 이용해 해결

**데이터 퍼짐이 작은 현상**  
거의 없으나 물체의 운동이 예시가 됨-> 미분방정식을 이용한 예측

## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
* 기술통계와 추론통계의 차이점을 설명할 수 있다.
* 통계적 추론과 가설 검정에 대해 이해한다.
```

**기술통계와 추론통계**
- 기술통계: 데이터 자체를 정리 요약
  <br>
- 추론통계: 데이터의 발생원을 추정하는 방법  
  확률모형: 비교적 단순한 확률 장치 ex) 주사위(1-6)  
  <br>
  통계적추론: 데이터에서 가정한 확률모형의 성질을 추정  
  가설 검정: 세운 가설과 얻은 데이터가 얼마나 들어맞는지 평가->가설 채택 여부 결정

**다양한 분석 방법**  
데이터의 유형이나 변수의 개수, 가정하는 확률 모형에 따라 다양한 방법이 존재

## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```

**모집단을 생각하다**  
- 모집단: 알고자 하는 대상 전체  
주의) 데이터를 얻을 가능성이 없는 요소는 제외한다  


**모집단 크기**  
- 모집단 크기: 모집단에 포함된 요소 수  
유한모집단: 한정된 요소만 포함 ex) 한국인...엄밀히 말하면 애매함  
무한모집단: 요소의 개수가 무한 ex) 신약의 효과 대상

## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```

**모집단의 성질**  
모집단의 성질을 알 수 있다면 대상 이해, 예측 가능  

**전수조사**  
- 전수 조사: 모집단에 모든 요소 조사  
유한모집단에서만 가능, 기술통계

- 전수조사는 비용, 시간 면에서 부담이 커 실현 불가능한 경우가 많음  

**표본조사**  
- 추론통계(inferential statistics): 모집단의 일부를 분석하여 모집단 전체의 성질을 추정  
- 표본(sample): 모집단의 일부  
- 표본추출(sampling): 모집단에서 표본을 뽑는 것  
- 표본조사: 표본을 이용하여 모집단 조사
<br>
불확실성을 염두에 두어야함  

**표본크기**  
- 표본크기: 표본의 포함된 요소의 개수, 주로 n으로 표기  
  혼동 주의) 샘플수(표본의 개수): 군집의 수
  <br>
  '표본의 크기는 모집단의 성질을 추정할 때의 확실성이나 가설 검정의 결과에 영향을 줌'

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~

```
정답: 1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
추론통계의 첫 단계는 표본추출 단계이다.
이를 이용하여 설문을 진행하고 평균을 내고 표본조사를 하는 과정은 그 이후에 진행된다.
50명이 모집단인 전체 직원 1,000명의 특성을 잘 드러낼 수 있는지 적절한 산포를 가지고 있는지 확인할 필요가 있다.
예를 들어 남녀 성비가 고른 모집단에서 남성의 표본만을 추출한다면 이는 모집단을 반영한 표본이라고 할 수 없다.
```
